# Koledar

Koledar vsebuje:

* Dnevne dogodke
* Obiski enote
* Najemi prostorov
* Odsotnosti zaposlenih
* Informacija o doseženi kvoti odsotnih (vodje, vozniki, dispečerji, potapljači, gasilci....)
* Prazniki

{% hint style="info" %}
Kaj vidi zaposleni na koledarju, je odvisno od dodeljenih pravic zaposlenega.
{% endhint %}
